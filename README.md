# 一分钟联盟营销系统

这是一个本地可运行的联盟营销系统原型，系统按「充值配套」驱动积分、推荐权限和联盟奖励。

## 5 个模块

1. 用户系统
   - 注册用户
   - 手机号或邮箱
   - 用户邀请码
   - 推荐人绑定
   - 用户等级
   - 配套状态

2. 支付积分系统
   - 充值配套
   - 积分发放
   - 积分余额
   - 积分流水
   - 配套有效期

3. 推荐权限系统
   - 配套发放推荐名额
   - 推荐关系固定
   - 推荐名额只限制直接推荐人数
   - 复购收益不消耗推荐名额

4. 联盟奖励系统
   - 首充奖励
   - 复购奖励
   - 待确认、已确认、已取消、已冻结
   - 订单完成 7 天后才可确认奖励
   - 推荐人配套过期时停止未来复购奖励

5. 后台管理系统
   - 查看用户
   - 查看订单
   - 修改积分
   - 查看推荐关系
   - 查看奖励记录
   - 冻结用户
   - 手动审核奖励
   - 设置配套规则
   - 设置奖励比例
   - 导出数据

## 默认配套示例

系统已内置 RM180 启动配套：

- 充值金额：RM180
- 发放积分：18,000
- 推荐权限：10 个
- 配套有效期：30 天
- 首充奖励：20%
- 复购奖励：8%

## 使用方式

直接打开 `index.html` 即可使用。当前版本的数据保存在浏览器 `localStorage` 中，适合演示、内测和确认业务规则。

正式上线前建议继续接入：

- 登录权限
- 数据库
- 支付网关回调
- 提现申请
- 财务审核
- 风控规则
- 管理员账号权限
