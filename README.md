# 一分钟联盟营销系统

这是一个本地可运行的双界面联盟营销系统原型。

## 用户界面

用户端用于普通会员操作：

- 登录或注册
- 输入推荐码绑定推荐人
- 申请充值配套
- 查看积分余额和配套状态
- 复制推荐链接
- 查看直接推荐朋友
- 查看首充奖励和下线复购奖励
- 查看奖励说明和奖励计算
- 申请奖励提现

当前奖励只做单层：

- 首充奖励：直接下线第一次购买配套时产生
- 下线复购奖励：直接下线再次购买配套时产生
- 推荐人配套过期后，不再产生未来复购奖励

## 后台界面

后台端用于管理员操作：

- 设置充值配套
- 设置积分、推荐名额、有效期
- 设置首充奖励比例
- 设置下线复购奖励比例
- 查看用户
- 冻结或解冻用户
- 修改用户积分
- 查看订单
- 审核奖励
- 审核提现
- 导出数据

## 关键规则

- 推荐关系固定，注册绑定后不允许随便更换。
- 奖励默认进入待处理状态。
- 订单完成 7 天后，后台才可以确认奖励。
- 已确认奖励才可以申请提现。
- 提现申请需要后台审核。
- 推荐名额只限制直接推荐人数，不限制这些下线未来复购次数。

## 使用

直接打开 `index.html` 即可使用。

当前版本数据保存在浏览器 `localStorage`，适合演示和确认业务流程。正式上线需要继续接入数据库、登录权限、支付网关、提现打款和管理员权限。
